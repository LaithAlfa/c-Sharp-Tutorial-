#Git Push to github

#...create a new repository on the command line
--------------------------------------------------------------------------

*git init

*git add README.md

*git commit -m "first commit"

*git branch -M main

*git remote add origin 

*git push -u origin main



#push an existing repository from the command line
----------------------------------------------------------

*git remote add origin

*git branch -M main

*git push -u origin main
